visual에서 수정 후 github에 수정본 보내는 법

1. 수정 파일 visual에 있는 터미널에 git add . 작성.

2. git status 로 파일이 수정 되었는지 확인.

3. git commit -m "second commit" 작성.

(여기서 큰 따옴표 안에는 히스토리 이름을 작성 ex> 세번 째 파일일 경우: third commit)

예를들어 히스토리 이름이 second commit 이면 앞에 second는 바뀌어도 되지만 뒤에 commit은 필수적으로 들어가야 한다.

4. git push origin master 를 작성.

==> visual에 작성한 코드 github으로 보내기 성공
